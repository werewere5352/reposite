Tools

all'interno di questa sezione verranno riportati una serie di tools per la risoluzione di CTF per la cybersicurezza con tanto di descrizione

  • dCode >>> Tool molto utile per crittografia e codifica/decodifica di parole o frasi
  • quipquip >>> simile a dCode ma con la differenza che si puo impostare una parte di testo gia conosciuta.
  • AperiSolve >>> AperiSolve serve per analizzare i metadati di un'immagine.
  • CyberChef >>> il piu famoso tool per la codifica/decodifica di testi fra i vari linguaggi(hex,base64,binary,decimal...)
  • HexEdit >>> Serve per modificare i metadati di qualsiasi tipo di file(.mp3,.jpg,.png...)
  • GitHub >>> non è un tool bensì GitHub è una piattaforma che permette di gestire, condividere e collaborare su progetti di sviluppo software.
  • WireShark >>> Serve per cattura e analisi rete ,tutorial.
  • BurpSuite >>> Utilizzato principalmente per identificare vulnerabilità come SQL injection, cross-site scripting (XSS) e altre falle di sicurezza,tutorial completo.
  • Git >>> Git è uno strumento per tenere traccia, gestire e collaborare sulle modifiche al codice in modo sicuro e ben organizzato, tutorial.
  • XAMPP >>> Software che fornisce un ambiente di sviluppo locale per creare, testare e implementare applicazioni web , tutorial.

CTF

Le CTF (Capture The Flag) sono delle sfide di sicurezza informatica nelle quali si affrontano sfide per mettere alla prova e migliorare le proprie competenze tecniche. L'obiettivo è scoprire delle "flag" (stringhe di testo nascoste), risolvendo problemi di crittografia, reverse engineering, sicurezza web e molto altro.

Tipi di CTF

  • Jeopardy: >>> una serie di challenge con punteggi che variano in base alla difficoltà
  • Attack/Defense: Le squadre difendono i propri sistemi da attacchi degli avversari, mentre cercano di compromettere quelli degli altri per ottenere flag.
  • Boot2Root: L'obiettivo è ottenere l'accesso root a una macchina vulnerabile, sfruttando falle presenti nel sistema.

Dove farle?

  • Piattaforme >>> Siti interessanti dove farle sono OliCyber, Hack The Box, TryHackMe o PicoCTF.
  • Eventi >>> eventi importanti per le ctf sono olicyber , CTFtime, DEFCON o PICOCTF.
